# Google Apps Scriptで作る！平日最終日自動設定ツール

月末の作業やレポート提出など、**平日最終日（月末営業日）** に予定を入れたいことってありませんか？

Googleカレンダーには平日最終日を自動設定する機能がないため、毎月手動でカレンダーを確認して登録する必要がありました。

そこで今回は、**Google Apps Scriptを使って平日最終日に自動でイベントを一括登録できるWebアプリ**を作成しました！

![スクリーンショット 2025-09-03 215004.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/2433285/f8b465da-88c4-4cb4-876c-8745fee81b8f.png)


## このツールで解決できる課題

### 従来の問題
- 毎月手動で月末営業日を計算する必要がある
- 複数月分の予定を一つずつ登録する手間
- 土日祝日を避けた最終営業日の把握が面倒
- 定期的な月末作業の予定管理が煩雑

### このツールの解決策
- **平日最終日の自動検出** - 土日を自動で除外した月末営業日を計算
- **期間指定での一括登録** - 複数月分を一度に設定
- **テンプレート機能** - よく使うイベントパターンを保存・再利用
- **プレビュー機能** - 登録前に対象日付を確認

## 主な機能

### 🎯 コア機能
- **平日最終日の自動検出**: 各月の土日を除いた最終営業日を自動計算
- **期間指定登録**: 開始月〜終了月の範囲で一括設定
- **個別月選択**: 今後12ヶ月から必要な月のみを選択
- **重複防止**: 既存イベントがある場合は自動スキップ

### 🛠️ 便利機能
- **イベントテンプレート**: よく使う設定パターンの保存・再利用
- **カラー設定**: 11色から選択してカレンダー上で視覚的に分類
- **時間指定対応**: 終日イベントと時間指定イベントの両方に対応
- **プレビュー表示**: 登録前に対象日付を確認
- **詳細レポート**: 登録件数、スキップ件数、エラー情報を表示

### 📱 UI/UX
- **レスポンシブデザイン**: スマホでも使いやすいモダンなUI
- **直感的な操作**: 段階的な設定フローで迷わない
- **リアルタイム検証**: カレンダーアクセス権限の事前テスト

### Code.gs（Google Apps Script側）

```javascript
/**
 * 平日最終日設定アプリ - Google Apps Script
 * 指定した期間の平日最終日にイベントを一括登録するツール
 */

/**
 * ウェブアプリケーションのHTMLファイルを提供します。
 */
function doGet() {
  return HtmlService.createTemplateFromFile('Index').evaluate()
    .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);
}

/**
 * 現在のユーザーのメールアドレスと保存済みイベント情報を取得します。
 */
function getCurrentUserEmailAndEventTemplates() {
  try {
    var userEmail = Session.getActiveUser().getEmail();
    var eventTemplates = [];
    
    if (userEmail) {
      var userProperties = PropertiesService.getUserProperties();
      var storedTemplatesJson = userProperties.getProperty('savedEventTemplates');
      if (storedTemplatesJson) {
        eventTemplates = JSON.parse(storedTemplatesJson);
      }
    }
    
    return { 
      userEmail: userEmail, 
      eventTemplates: eventTemplates 
    };
  } catch (e) {
    Logger.log('ユーザー情報取得エラー: ' + e.message);
    return { 
      userEmail: Session.getActiveUser().getEmail() || "", 
      eventTemplates: [] 
    };
  }
}

/**
 * 指定した期間の平日最終日を取得します。
 */
function getWeekdayFinalDates(startMonth, endMonth) {
  try {
    var startDate = new Date(startMonth + '-01');
    var endDate = new Date(endMonth + '-01');
    
    // 終了月の最終日を設定
    endDate.setMonth(endDate.getMonth() + 1);
    endDate.setDate(0);
    
    var weekdayFinalDates = [];
    var currentDate = new Date(startDate);
    
    while (currentDate <= endDate) {
      // その月の最終日を取得
      var lastDayOfMonth = new Date(currentDate.getFullYear(), currentDate.getMonth() + 1, 0);
      
      // 平日最終日を見つける（土日を除外）
      while (lastDayOfMonth.getDay() === 0 || lastDayOfMonth.getDay() === 6) {
        lastDayOfMonth.setDate(lastDayOfMonth.getDate() - 1);
      }
      
      // YYYY-MM-DD形式で追加
      var dateString = lastDayOfMonth.getFullYear() + '-' + 
                      String(lastDayOfMonth.getMonth() + 1).padStart(2, '0') + '-' + 
                      String(lastDayOfMonth.getDate()).padStart(2, '0');
      
      weekdayFinalDates.push(dateString);
      
      // 次の月へ
      currentDate.setMonth(currentDate.getMonth() + 1);
    }
    
    return weekdayFinalDates;
  } catch (e) {
    Logger.log('平日最終日計算エラー: ' + e.message);
    throw new Error('平日最終日の計算中にエラーが発生しました: ' + e.message);
  }
}

/**
 * 指定した平日最終日にイベントを一括登録します。
 */
function addEventOnWeekdayFinals(calendarId, startMonth, endMonth, eventTitle, isAllDay, startTimeStr, endTimeStr, eventColor) {
  try {
    var calendar = CalendarApp.getCalendarById(calendarId);
    if (!calendar) {
      throw new Error('指定されたカレンダーが見つかりません: ' + calendarId);
    }
    
    // 平日最終日を取得
    var weekdayFinalDates = getWeekdayFinalDates(startMonth, endMonth);
    var addedCount = 0;
    var skippedCount = 0;
    var addedDates = [];
    
    weekdayFinalDates.forEach(function(dateString) {
      var baseDate = new Date(dateString);
      
      // 既存イベントの重複チェック
      var searchStartTime = new Date(baseDate);
      searchStartTime.setHours(0, 0, 0, 0);
      var searchEndTime = new Date(baseDate);
      searchEndTime.setHours(23, 59, 59, 999);
      
      var existingEvents = calendar.getEvents(searchStartTime, searchEndTime, { search: eventTitle });
      var eventExists = existingEvents.some(function(event) {
        return event.getTitle() === eventTitle;
      });
      
      if (!eventExists) {
        var eventStart, eventEnd, newEvent;
        
        if (isAllDay) {
          newEvent = calendar.createAllDayEvent(eventTitle, baseDate);
        } else {
          var startTimeParts = startTimeStr.split(':');
          var endTimeParts = endTimeStr.split(':');
          
          eventStart = new Date(baseDate);
          eventStart.setHours(parseInt(startTimeParts[0]), parseInt(startTimeParts[1]), 0, 0);
          
          eventEnd = new Date(baseDate);
          eventEnd.setHours(parseInt(endTimeParts[0]), parseInt(endTimeParts[1]), 0, 0);
          
          if (eventEnd.getTime() < eventStart.getTime()) {
            eventEnd.setDate(eventEnd.getDate() + 1);
          }
          
          newEvent = calendar.createEvent(eventTitle, eventStart, eventEnd);
        }
        
        // イベント色を設定
        if (eventColor && isValidEventColor(eventColor)) {
          try {
            newEvent.setColor(CalendarApp.EventColor[eventColor]);
          } catch (colorError) {
            Logger.log('色設定失敗: ' + colorError.message);
          }
        }
        
        addedCount++;
        addedDates.push(dateString);
      } else {
        skippedCount++;
      }
    });
    
    return {
      success: true,
      addedCount: addedCount,
      skippedCount: skippedCount,
      addedDates: addedDates,
      message: addedCount + '件のイベントを平日最終日に登録しました。' + 
               (skippedCount > 0 ? ' (' + skippedCount + '件は既存のためスキップ)' : '')
    };
    
  } catch (e) {
    Logger.log('平日最終日イベント登録エラー: ' + e.message);
    return {
      success: false,
      message: 'イベント登録中にエラーが発生しました: ' + e.message
    };
  }
}

/**
 * 有効なイベント色かどうかをチェック
 */
function isValidEventColor(eventColor) {
  var validColors = [
    'LAVENDER', 'SAGE', 'PURPLE', 'PINK', 'YELLOW', 
    'ORANGE', 'CYAN', 'GREY', 'BLUE', 'GREEN', 'RED'
  ];
  return validColors.indexOf(eventColor) !== -1;
}
```

## セットアップ方法

### 1. Google Apps Scriptプロジェクトの作成

1. [Google Apps Script](https://script.google.com/) にアクセス
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「平日最終日設定アプリ」に変更

### 2. ファイルの追加

**Code.gsファイル:**
- デフォルトのCode.gsに上記のGoogle Apps Scriptコードを貼り付け

**Index.htmlファイル:**
1. 「ファイル」→「新規」→「HTMLファイル」を選択
2. ファイル名を「Index」に設定
3. 上記のHTMLコードを貼り付け

### 3. 権限とデプロイ設定

**必要な権限:**
- Googleカレンダーへの読み書き権限
- ユーザープロパティへのアクセス権限

**デプロイ手順:**
1. 「デプロイ」→「新しいデプロイ」を選択
2. 種類を「ウェブアプリ」に設定
3. 実行ユーザーを「自分」に設定
4. アクセス権限を「自分のみ」または「Googleアカウントを持つ全員」に設定
5. 「デプロイ」をクリック

## 使い方

### 基本的な使用手順

1. **カレンダーIDの入力**
   - 自分のGoogleアカウントのメールアドレスを入力
   - 「テスト」ボタンでアクセス権限を確認

2. **期間の選択**
   - **期間指定モード**: 開始月〜終了月を指定
   - **個別選択モード**: 今後12ヶ月から必要な月のみを選択

3. **平日最終日の確認**
   - 「プレビュー表示」で対象となる日付を事前確認

4. **イベント詳細の設定**
   - イベント名を入力
   - 終日イベントか時間指定かを選択
   - 必要に応じてイベント色を設定

5. **一括登録の実行**
   - 「平日最終日にイベント登録」ボタンで一括実行

## 技術的なポイント

### 平日最終日の計算ロジック

```javascript
// その月の最終日を取得
var lastDayOfMonth = new Date(year, month + 1, 0);

// 土日を除外して平日最終日を取得
while (lastDayOfMonth.getDay() === 0 || lastDayOfMonth.getDay() === 6) {
  lastDayOfMonth.setDate(lastDayOfMonth.getDate() - 1);
}
```

このロジックにより、各月の最終日から遡って最初の平日（月曜〜金曜）を自動検出します。
