## 1. Claude Code Webとは？

Claude Code Webは、Anthropic社が提供する**AIプログラミングアシスタント**です。

### できること

- GitHubリポジトリに直接アクセス
- コードの作成・修正・削除
- ドキュメントの整理
- プルリクエストの自動作成
- Issue管理のサポート

### 特徴

- **ブラウザだけで完結**（ソフトウェアのインストール不要）
- **自然言語で指示**できる（プログラミング知識不要）
- **クラウド実行**なので、PCのスペックに依存しない

---

## 2. なぜモバイルから使えることが重要なのか

Claude Code Webはスマホのブラウザから操作できます。

### 従来の開発環境との違い

| 従来の開発環境 | Claude Code Web |
|--------------|-----------------|
| PCが必須 | スマホだけで操作可能 |
| IDEのインストールが必要 | ブラウザだけで動作 |
| PCの電源が切れたら作業停止 | クラウド実行のため継続 |
| 座って作業する必要がある | 立ったままでも操作可能 |
| 移動中は作業できない | 移動中でも操作可能 |

### スキマ時間の活用

まとまった時間が必要だったコード整理が、短時間でできます。

- 通勤電車（15分）: READMEの整理
- ランチ後の休憩（10分）: 不要ファイルの削除
- 散歩中（5分）: Issueの整理

### 思いついた時に実行

作業を後回しにせず、その場で依頼できます。

- カフェで思い出した → その場で整理依頼
- 散歩中に気づいた → 歩きながら更新依頼
- 電車で思い出した → その場で指示

### PCを起動する手間がない

スマホでブラウザを開くだけで作業を開始できます。指示を出したら閉じても、バックグラウンドで実行されます。

## 3. 必要なアカウントを準備する

Claude Code Webを使うには、以下の2つのアカウントが必要です。

### 3-1. GitHubアカウント（無料）

GitHubは、コードを保存・管理するサービスです。

**登録手順：**

1. [github.com](https://github.com) にアクセス
2. 「Sign up」をクリック
3. メールアドレス・パスワード・ユーザー名を入力
4. 認証メールを確認して完了

### 3-2. Claudeアカウント（有料：月額$20〜）

Claude Code Webは**有料プラン限定**の機能です。

| プラン | 月額 | 日本円換算 | 推奨用途 |
|--------|------|-----------|----------|
| **Pro** | $20 | 約3,000円 | 個人開発・学習向け（推奨） |
| Max 5x | $100 | 約15,000円 | 本格開発向け |
| Max 20x | $200 | 約30,000円 | ヘビーユーザー向け |

**登録手順：**

1. [claude.ai](https://claude.ai) にアクセス
2. 「Sign up」でアカウント作成
3. ログイン後、左下の「**Upgrade to Pro**」をクリック
4. 支払い情報を入力して有料プランに登録

> ⚠️ **重要**: 無料プランではClaude Code Webは使えません。最低でもProプラン（$20/月）が必要です。

## 4. Claude Code WebとGitHubを連携する

ここからが本番です。Claude Code WebがあなたのGitHubリポジトリにアクセスできるように設定します。

### Step 1: Claude Code Webにアクセス

1. ブラウザで **https://claude.ai/code** にアクセス
2. Claudeアカウントでログイン
3. 「コード」画面が表示されることを確認

### Step 2: GitHubに接続

1. 画面に表示される「**GitHubに接続**」ボタンをクリック
2. GitHubのログイン画面が表示されたらログイン
3. 「**Authorize Claude**」をクリックして接続を許可

### Step 3: Claude GitHubアプリをインストール

1. 「**Claude GitHubアプリをインストール**」ボタンをクリック
2. アクセス許可画面が表示される
3. リポジトリの公開範囲を選択：
   - 「**Only select repositories**」：特定のリポジトリのみ許可（推奨）
   - 「All repositories」：すべてのリポジトリを許可
4. 「**Install & Authorize**」をクリック

> 💡 **セキュリティのポイント**: 最初は「Only select repositories」で1〜2個のテスト用リポジトリだけを選択するのがおすすめです。

### Step 4: 作業するリポジトリを選択

1. Claude Code Webの画面に戻る
2. 連携したリポジトリの一覧が表示される
3. 整理したいリポジトリを選択

これで準備完了です！

## 5. スマホからアクセスする方法

PCは不要です。スマホのブラウザだけで操作できます。

### アクセス手順

iPhone・Androidどちらも同じ手順です。

1. SafariまたはChromeを起動
2. https://claude.ai/code にアクセス
3. Claudeアカウントでログイン
4. 作業したいリポジトリを選択

アプリのインストールは不要です。

### ブックマーク

頻繁に使う場合は、`https://claude.ai/code` をブックマーク（お気に入り）に追加すると便利です。

**iPhoneの場合：**
1. Safariで https://claude.ai/code を開く
2. 下部の「共有」ボタン→「ホーム画面に追加」
3. ホーム画面から起動できる

**Androidの場合：**
1. Chromeで https://claude.ai/code を開く
2. 右上の「︙」→「ホーム画面に追加」
3. ホーム画面から起動できる

### PCが不要な理由

Claude Code WebはAnthropicのクラウドで動作します。

- スマホやPCは指示を送るだけ
- 実際のコード実行はクラウド上で行われる
- デバイスのスペックは関係ない
- タスク実行中にブラウザを閉じても作業は継続

例：
```
通勤電車で「README整理して」と指示
→ ブラウザを閉じる
→ 10分後に通知が来る
→ 再度アクセスして結果を確認
```

### 複数タスクの同時実行

Claude Code Webは、複数のリポジトリで同時に作業できます。

```
リポジトリA: README整理（実行中）
リポジトリB: 不要ファイル削除（実行中）
リポジトリC: 依存関係更新（実行中）
```

すべてクラウドで並列実行されるため、スマホに負荷はかかりません。

## 6. リポジトリ整理を依頼してみる

いよいよ実際にClaude Code Webに整理作業を依頼してみましょう。

### 6-1. 基本的な依頼の仕方

Claude Code Webの入力欄に、普通の日本語で指示を書くだけです。

**入力例：**
```
このリポジトリのREADMEを見直して、
分かりやすく整理してください。
```

「送信」ボタンを押すと、Claudeが：
1. READMEファイルを確認
2. 改善案を提案
3. 修正を実行
4. 結果を報告

してくれます。

### 6-2. よくあるリポジトリ整理の依頼例

#### ドキュメント整理

```
READMEを以下の点で改善してください：
- 古い情報を削除
- インストール手順を最新版に更新
- 目次を追加
- 使い方のセクションを見やすく整理
```

#### 不要ファイルの削除

```
このリポジトリで不要なファイルを探して削除してください。
具体的には：
- node_modules やビルド生成物
- .DS_Store などのOSファイル
- 古いログファイル

.gitignore も適切に更新してください。
```

#### コメント追加

```
src/以下のすべてのJavaScriptファイルに、
分かりやすい日本語コメントを追加してください。
特に関数の役割と引数の説明を丁寧にお願いします。
```

#### Issue整理の手伝い

```
このリポジトリのIssueを確認して、
以下の情報をまとめてください：
- 対応済みだがクローズされていないIssue
- 重複しているIssue
- 6ヶ月以上更新がないIssue
```

#### 依存関係の確認

```
package.jsonの依存関係を確認して、
以下を教えてください：
- セキュリティ脆弱性があるパッケージ
- 長期間更新されていないパッケージ
- 使われていない依存関係

安全に更新できるものがあれば、更新案を提案してください。
```

#### ディレクトリ構造の改善

```
このプロジェクトのディレクトリ構造を見直して、
より標準的な構造に再編成してください。
変更内容を実行前に提案してください。
```

### 6-3. プルリクエストの作成を依頼する

変更をmainブランチに直接反映せず、プルリクエスト（PR）として提案してもらうこともできます。

```
READMEを整理して、プルリクエストを作成してください。
PR のタイトルは「READMEの改善」でお願いします。
```

こうすると：
1. Claudeが新しいブランチを作成
2. そのブランチで修正を実施
3. GitHubにプッシュ
4. プルリクエストを自動作成

してくれます。あなたは GitHub でPRを確認して、問題なければマージするだけです。

## 7. モバイルならではの活用シーン

スマホから使える特性を活かした使い方の例です。

### 通勤電車でリポジトリ整理

**所要時間**: 片道30分

**手順**:
```
1. Claude Code Webを開く
2. 「このリポジトリの不要ファイルを削除して、
   .gitignoreも適切に更新してください」と入力
3. 送信後はブラウザを閉じる
4. 到着後に通知を確認
5. 会社のPCで結果を確認してマージ
```

### ランチ後の休憩時間

**所要時間**: 5〜10分

**手順**:
```
1. スマホを開く
2. 「READMEのインストール手順を
   最新版に更新してください」
3. 数分後に確認してマージ
```

### 週末の散歩中

**所要時間**: 歩きながら数分

**手順**:
```
1. 歩きながらスマホで指示
   「package.jsonの依存関係を確認して、
   セキュリティ脆弱性があるパッケージを
   リストアップしてください」
2. 帰宅後に結果を確認
```

### 寝る前

**所要時間**: 数分

**手順**:
```
1. 「6ヶ月以上更新がないIssueをリストアップして、
   クローズできそうなものを提案してください」
2. 翌朝、通勤中に結果を確認
```

### 旅行先からの対応

**状況**: 週末の旅行中、急な対応が必要になった場合

**手順**:
```
1. スマホでClaude Code Webにアクセス
2. 「ログイン機能でエラーが発生しています。
   原因を調査して修正案を提案してください」
3. 提案を確認してPR作成を依頼
```

### 待ち時間の活用

**状況**: 1時間程度の待ち時間がある場合

**手順**:
```
1. 複数のリポジトリに指示
   - リポジトリA: README整理
   - リポジトリB: 不要ファイル削除
   - リポジトリC: コメント追加
2. 30分後に確認
3. 問題なければマージ
```

### 使い方のコツ

**指示は短く具体的に**

スマホでの入力は短い方が楽です。
```
良い例: 「READMEを整理してPR作成」
```

**結果の確認は後でもOK**

タスク実行中にブラウザを閉じても作業は継続します。
```
朝の通勤: 指示を出す
昼休み: 結果を確認
帰宅後: マージ
```

**複数タスクを同時に進める**

複数リポジトリに同時に指示を出せます。
```
リポジトリA、B、Cに同時に指示
→ まとめて確認
```

## 8. よくある質問

### Q1. 無料で使えますか？

A. Claude Code Webは有料プラン限定の機能です。最低でも**Proプラン（月額$20/約3,000円）** が必要です。

### Q2. スマホからも使えますか？

A. はい、スマホのブラウザからも使えます。**https://claude.ai/code** にアクセスすればOKです。

### Q3. プライベートリポジトリも使えますか？

A. はい、プライベートリポジトリでも使えます。GitHubアプリのインストール時に対象リポジトリを選択できます。

### Q4. 間違った変更をされたらどうなりますか？

A. GitHubの履歴（コミット履歴）が残っているので、いつでも元に戻せます。心配な場合は、まずテスト用のブランチで試してみることをおすすめします。

### Q5. どんなリポジトリでも整理できますか？

A. Claude Code Webは多くのプログラミング言語に対応していますが、特に以下の言語で高い精度を発揮します：

- JavaScript / TypeScript
- Python
- React / Next.js
- HTML / CSS
- Markdown

### Q6. プログラミングの知識は必要ですか？

A. 基本的には不要です。普通の日本語で指示を出すだけで、Claudeが適切にコードを理解して作業してくれます。

ただし、以下のような最低限の知識があると、より効果的に使えます：

- GitHubの基本的な仕組み（リポジトリ、ブランチ、プルリクエスト）
- 整理したいファイルの役割（README、.gitignoreなど）

### Q7. 費用はClaude Proの月額だけですか？

A. はい、GitHubは無料プランで十分使えます。必要な費用は**Claude Pro（月額$20/約3,000円）のみ**です。
