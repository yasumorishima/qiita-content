### Webページの全ての「Add」ボタンを一括でクリックするブックマークレット

Webサイトを使っていて、たくさんの「追加」ボタンを一つ一つ手動でクリックするのは面倒な作業です。特に、一覧から多くの項目を追加する必要がある時、この作業は時間の無駄に感じられます。そんな時に便利なのが、このブックマークレットです。

#### このブックマークレットは何をするのか？

このブックマークレットは、ページ上の全ての「Add」ボタンを自動でクリックします。「Delete」ボタンがある場合はそれを無視するため、既に追加されている項目を誤って削除する心配はありません。

#### 使い方

1.  以下のJavaScriptコードをコピーします。

```javascript
javascript:(function() {
  document.querySelectorAll('button').forEach(function(btn) {
    if (btn.innerHTML.includes('AddIcon') && !btn.disabled) {
      btn.click();
    }
  });
})();
```

2.  ブラウザでブックマークバーを表示させます。
3.  ブックマークバー上で右クリック（またはコントロールクリック）し、「新しいブックマーク」または「ブックマークを追加」を選択します。
4.  名前フィールドに、ブックマークレットにわかりやすい名前を入力します（例: 「全Addボタンをクリック」）。
5.  URLフィールドに、コピーした`javascript:`で始まるコードをペーストします。
6.  保存します。

これで、使用したいページでこのブックマークをクリックするだけで、ページ上の全ての「Add」ボタンが自動的にクリックされます。

#### 注意点

*   このブックマークレットは、HTML構造やクラス名が予期した通りであるページでのみ正しく動作します。
*   セキュリティの観点から、知らないソースからのスクリプトは実行しないようにしてください。
*   特定のサイトでの使用においては、サイトの利用規約に違反しないよう注意してください。

この小さなツールが、日々の作業を少しでも楽にしてくれることを願っています。
