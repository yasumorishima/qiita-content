これから記載の内容は、以下です。

1.  NFT取引データをリポジトリから取得し、必要なライブラリをインストールする。
2.  取得したデータをPandasデータフレームに読み込む。
3.  Seabornを使用して、取引データを散布図で可視化する。
4.  最終日の最小価格取引データを抽出し、整形して表示する。
5.  最終日の最小価格取引データを11行ごとに分割し、整形して表示する。

全体の流れとして、まずNFT取引データを取得し、データを分析して可視化します。その後、最終日の最小価格取引データを抽出し、整形して表示します。最後に、データを11行ごとに分割し、整形して表示します。

# 最終日の最小価格取引データを表示

このコードには、最終日の最小価格取引データを表示するための処理が含まれています。具体的には、必要なカラムを抽出し、インデックスをリセットし、データフレームのカラム名にラベルを追加して表示しています。

このコードを実行すると、最終日の最小価格取引データが表示されます。データフレームには、「day」、「collection」、「icp」のカラムが含まれており、各行は最終日の各コレクションの最小価格取引データを示しています。カラム名にはラベル「最終日の最小価格取引(ICP)」が追加されています。

<pre><code class># 各コレクションの最終日を取得
last_days = df_min.reset_index().groupby('collection')['day'].max()

# 最終日のデータを抽出
last_day_data = df_min.reset_index()[df_min.reset_index().apply(lambda x: x['day'] == last_days[x['collection']], axis=1)]

# 結果を表示
print(last_day_data)

</code></pre>

# tabulate ライブラリを使って、最終日の最小価格取引データを整形して表示 (分割)

このコードは、最終日の最小価格取引データを11行ごとに分割し、それぞれの部分を整形して表示するためのスクリプトです。

このコードを実行すると、最終日の最小価格取引データが11行ごとに分割され、それぞれの部分が整形された表として表示されます。各表には、部分の番号が表示され、カラム名がヘッダーとして表示されます。表の書式は、グリッド線で区切られたものになっています。部分ごとに改行が挿入されています。

<pre><code class>from tabulate import tabulate

# 表を11行ごとに分割
n = 11
table_parts = [last_day_data.iloc[i:i+n] for i in range(0, len(last_day_data), n)]

# 各部分を表示
for i, part in enumerate(table_parts):
    print(f"Entrepot 取引があった最終日の最小価格取引(ICP) - Part {i+1}")
    print(tabulate(part, headers='keys', tablefmt='grid'))
    print("\n")

</code></pre>
