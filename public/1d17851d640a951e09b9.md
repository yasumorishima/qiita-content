# Google ColabでHTMLをPDFに変換する方法：日本語対応とトラブルシューティング

Google Colaboratory（Colab）は、Pythonのコードをブラウザから実行できる便利なクラウド環境です。しかし、Webページ（HTML）をPDFに変換するような、ブラウザのレンダリング機能に依存する処理を行う場合、Colab特有の環境設定が必要になります。

この記事では、SeleniumとChrome WebDriverを使って、HTMLファイルをPDFに変換する手順を詳しく解説します。特に、Colab環境でのセットアップ、日本語の文字化け対策、そしてよくあるトラブルシューティングについても触れます。

## なぜHTMLをPDFに変換するのか？

HTMLをPDFに変換する用途は多岐にわたります：

- **動的コンテンツの保存**：JavaScriptで生成されるグラフやインタラクティブな要素を含むWebページを、見た目そのままに保存したい場合
- **オフラインでの閲覧**：インターネット接続がない環境でもWebページの内容を確認したい場合
- **資料作成**：Webサイトの情報をレポートやプレゼンテーションに組み込みたい場合
- **テキスト検索・選択**：画像として保存するのではなく、PDFとしてテキスト情報を保持したい場合

## 変換プロセスの概要

今回紹介する変換プロセスは、以下のステップで構成されます：

1. **環境セットアップ**：Google ColabにChromeブラウザ、WebDriver、日本語フォントをインストールします
2. **HTMLファイルのアップロード**：変換したいHTMLファイルをColab環境にアップロードします
3. **PDF変換の実行**：Seleniumを使ってHTMLをChromeでレンダリングし、PDFとして出力します
4. **PDFファイルのダウンロード**：生成されたPDFファイルをColabからローカルPCにダウンロードします

## 詳細手順とコード

Google Colabのノートブックで、以下の各コードブロックを上から順番に実行してください。

### ステップ1：環境セットアップ（Chrome、WebDriver、日本語フォント）

Colabの初期環境には、Chromeブラウザや WebDriver、日本語フォントがインストールされていません。これらをインストールし、Seleniumが正しく動作するようにパスを設定します。

```python
import time
import os
import io
import base64
import threading
from selenium import webdriver
from selenium.webdriver.chrome.options import Options as ChromeOptions
from selenium.webdriver.chrome.service import Service as ChromeService
from webdriver_manager.chrome import ChromeDriverManager
from google.colab import files

# Chromeブラウザのインストール
# Googleの公式ダウンロードリンクから.debパッケージを直接ダウンロードし、インストールします。
print("🔧 Chromeブラウザをインストール中...")
os.system("wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb")
os.system("dpkg -i google-chrome-stable_current_amd64.deb")
os.system("apt-get install -f -y")  # 依存関係を解決

# SeleniumとWebDriver Manager、base64ライブラリのインストール
print("📦 必要なライブラリをインストール中...")
os.system("pip install selenium webdriver-manager base64")

# 日本語フォントのインストール（IPAexフォント）
# PDFでの日本語文字化けを防ぐために非常に重要です。
print("🈳 日本語フォントをインストール中...")
os.system("apt-get install -y fonts-ipaexfont")

# フォントキャッシュの更新
print("🔄 フォントキャッシュを更新中...")
os.system("fc-cache -fv")

# Colab環境でChromeのパスをWebDriverに認識させるための設定
os.environ['PATH'] += ':/usr/bin'
os.environ['PATH'] += ':/usr/local/bin'

print("✅ Chromeブラウザ、WebDriver、日本語フォントのインストールが完了しました。")
```

### ステップ2：HTMLファイルのアップロード

変換したいHTMLファイルをローカルPCからColab環境にアップロードします。Colabは直接ローカルPCのファイルシステムにアクセスできないため、このアップロード手順が必要です。

```python
print("📁 PDFに変換したいHTMLファイルをアップロードしてください。")
uploaded = files.upload()

html_file_to_convert = None
for filename in uploaded.keys():
    html_file_to_convert = filename
    print(f"'{html_file_to_convert}' がアップロードされました。")
    break

if html_file_to_convert is None:
    print("⚠️ ファイルが選択されませんでした。処理を終了します。")
    raise Exception("No HTML file uploaded.")
```

### ステップ3：HTMLをPDFに変換するコードの実行

アップロードされたHTMLファイルをSeleniumを使ってChromeで開き、PDFとして出力します。

```python
def process_html_to_pdf(html_filepath, browser="chrome"):
    options_class = ChromeOptions
    driver_manager = ChromeDriverManager
    service_class = ChromeService

    browser_name = "Chrome"

    # 出力PDFファイル名の設定
    base_name = os.path.splitext(os.path.basename(html_filepath))[0]
    output_filename = f"{base_name}_full_page_{browser}.pdf"
    output_path = os.path.join(os.getcwd(), output_filename)
    
    # ブラウザオプションの設定
    browser_options = options_class()
    browser_options.add_argument('--headless')
    browser_options.add_argument('--no-sandbox')
    browser_options.add_argument('--disable-dev-shm-usage')
    browser_options.binary_location = '/usr/bin/google-chrome'
    
    driver = None
    try:
        print(f"🚀 WebDriverをセットアップ中 ({browser_name}版)...")
        
        try:
            service = service_class(ChromeDriverManager().install())
        except Exception as e:
            print(f"⚠️ webdriver_managerによるWebDriverの自動インストールに失敗しました: {e}")
            print("手動でWebDriverのパスを指定しようとします。")
            service = service_class("/usr/local/bin/chromedriver")
        
        driver = webdriver.Chrome(service=service, options=browser_options)
        print(f"🌟 {browser_name} WebDriverを起動しました。")
        
        # HTMLファイルをChromeで開く
        driver.get(f'file://{os.path.abspath(html_filepath)}')
        print(f"📖 HTMLファイル '{html_filepath}' を読み込み中...")
        
        # JavaScriptのレンダリングやフォントの読み込みが完了するまで待機
        time.sleep(10)
        
        # PDFとして印刷を実行（Chrome DevTools Protocolを使用）
        print_options = {
            'landscape': False,
            'displayHeaderFooter': False,
            'printBackground': True,
            'preferCSSPageSize': True,
            'scale': 1.0,
            'paperWidth': 8.27,    # A4サイズ
            'paperHeight': 11.69,  # A4サイズ
            'marginTop': 0,
            'marginBottom': 0,
            'marginLeft': 0,
            'marginRight': 0,
            'pageRanges': '',
            'headerTemplate': '<div></div>',
            'footerTemplate': '<div></div>'
        }
        
        # PDFデータをバイト列として取得
        pdf_bytes = driver.execute_cdp_cmd('Page.printToPDF', print_options)
        
        # PDFデータをファイルとして保存
        with open(output_path, "wb") as f:
            f.write(base64.b64decode(pdf_bytes['data']))
        
        print(f"🎉 HTMLがPDFとして '{output_path}' に保存されました。")
        global generated_pdf_path
        generated_pdf_path = output_path

    except Exception as e:
        print(f"❌ エラーが発生しました ({browser_name}版): {e}")
        print(f"PDF化中にエラーが発生しました。\n{e}")

    finally:
        if driver:
            driver.quit()
            print(f"🔴 {browser_name} WebDriverを終了しました。")

generated_pdf_path = None

# メイン処理の実行
if __name__ == "__main__":
    if 'html_file_to_convert' in locals() and html_file_to_convert:
        selected_browser = "chrome"
        processing_thread = threading.Thread(target=process_html_to_pdf,
                                            args=(html_file_to_convert, selected_browser))
        processing_thread.start()
        processing_thread.join()
        
        print("✅ PDF生成処理が終了しました。")
    else:
        print("⚠️ PDFに変換するHTMLファイルがアップロードされていません。")
```

### ステップ4：生成されたPDFファイルのダウンロード

Colab環境に保存されたPDFファイルを、ローカルPCのダウンロードフォルダにダウンロードします。

```python
if 'generated_pdf_path' in globals() and generated_pdf_path and os.path.exists(generated_pdf_path):
    print(f"📥 PDFファイルをダウンロードしています: {os.path.basename(generated_pdf_path)}")
    files.download(generated_pdf_path)
    print("✅ ダウンロードが完了しました。ローカルのダウンロードフォルダをご確認ください。")
else:
    print("⚠️ 生成されたPDFファイルが見つからないか、パスが設定されていません。")
    print("PDF生成が正常に完了したか確認してください。")
```

## トラブルシューティング

### 1.「Message: unknown error: cannot find Chrome binary」エラー

**原因**：Colab環境がChromeブラウザを見つけられない場合に発生します。

**解決策**：
- 「ステップ1：環境セットアップ」のコードがすべて正しく実行されているか確認してください
- 特に、`wget`と`dpkg -i`でChromeがインストールされ、`browser_options.binary_location = '/usr/bin/google-chrome'`で正しいパスが指定されていることが重要です
- Colabのセッションがリセットされた場合は、必ずステップ1からやり直してください

### 2. 日本語が文字化けする

**原因**：PDFに変換された日本語が正しく表示されない場合、フォントの問題が考えられます。

**解決策**：
- 「ステップ1：環境セットアップ」に`apt-get install -y fonts-ipaexfont`と`fc-cache -fv`が含まれていることを確認してください
- これにより、Colab環境に日本語フォントがインストールされ、ブラウザがそれを利用できるようになります
- また、HTMLファイル自体の文字エンコーディングがUTF-8であることも確認してください（`<meta charset="utf-8">`）

### 3. Colabのセッションがリセットされた場合

**原因**：Colabのランタイムが切断され、セッションがリセットされた場合は、すべての環境設定が失われます。

**解決策**：
- その場合は、必ずステップ1からやり直してください
- 環境セットアップには数分かかりますが、これを省略すると正常に動作しません
