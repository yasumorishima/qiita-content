# Google Apps Script - フォーム自動返信スクリプト完全ガイド

![スクリーンショット 2025-07-20 135404.jpg](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/2433285/9340277e-d051-4a54-ac30-090fecfd5f7f.jpeg)



## 概要

Google Apps Script（GAS）を使ってGoogleフォームの回答に基づいて自動返信メールを送信するスクリプトです。テストやアンケートの結果に応じて、合格・不合格メールを自動で送信できます。

## 背景と発生した問題

### 当初の問題
最初は `formResponse.getGradableResponse()` メソッドを使用していましたが、以下のエラーが発生しました：

```
formResponse.getGradableResponse is not a function
```

### 解決策
Google Workspace環境におけるセキュリティポリシーの制約により、フォームオブジェクトから直接回答を取得できない場合があります。この問題を回避するため、**Googleスプレッドシートから直接データを取得する方式**に変更しました。

## 完全なコード

```javascript
/**
 * @fileoverview
 * Googleフォームの回答に対して、教材の内容のメールを自動返信するスクリプトです。
 * - 90点以上の場合: 合格メールを送信
 * - 90点未満の場合: 合格したときに送るメール送信
 * - 全受験者: スクリプトからアップロード先を取得します。
 */

// --- 設定項目 ---
// 以下の設定値は環境に合わせて変更してください。

// 1. Googleフォームの回答が保存されているスプレッドシートのID
// スプレッドシートのURLから "d/"と"/edit"の間の文字列をコピーして設定してください。
const SPREADSHEET_ID = "ここにあなたのスプレッドシートIDを入力"; // 実際のIDに変更してください

// フォームの公開URL（推奨：直接指定）
const FORM_URL = "https://forms.gle/あなたのフォームURL"; // あなたのフォーム短縮URL

// 研修動画URL（不合格者向け）
const TRAINING_VIDEO_URL = "https://youtu.be/あなたの動画URL"; // 研修動画のURLを設定

// 2. フォームの回答が記録されるシート名（通常はフォーム回答1）
const SHEET_NAME = "フォームの回答 1"; // あなたのシート名に合わせて変更（実際の設定値はご確認ください）

// 3. タイムスタンプのインデックス（A列の）（通常は）
// あなたのスプレッドシートの構造に合わせて変更してください。
// 例: タイムスタンプ（A列）、メールアドレス（B列）、スコア（C列）の場合はスコア列は配列の2]
const SCORE_COLUMN_INDEX = 2; // C列のスコアの場合

// 4. 回答者のメールアドレスが記録されている列のインデックス
// あなたのスプレッドシートの構造に合わせて変更してください。
// 例: タイムスタンプ（A列）、メールアドレス（B列）、スコア（C列）の場合は、メールアドレス列は配列の[1]
const EMAIL_COLUMN_INDEX = 1; // B列のメールアドレスの場合

// 5. 合格点
const PASSING_SCORE = 90;

// 6. メールの件名
const EMAIL_SUBJECT = "【受講済】研修営業基礎テストの結果";

// --- 設定はここまで ---

/**
 * フォームの回答が保存されたときに実行されるトリガー関数
 * @param {GoogleAppsScript.Events.FormsOnFormSubmit} e - イベントオブジェクト（今回はobjは使用しないのでトリガーのみ利用）
 */
function onFormSubmit(e) {
  console.log("onFormSubmitトリガーが実行されました。");
  
  // スクリプトでテングのから毒を実行するため、e.responseは必須ではない。
  // スプレッドシートからデータを取得するため、トリガーは実行するタイミングの確認用です。
  
  if (!e || !e.response) {
    console.log("イベントオブジェクトまたはレスポンスが取得できませんでした。スプレッドシートから直接取得します。");
  }
  
  try {
    console.log("処理を開始します。");
    
    // Googleフォームの回答が保存されているスプレッドシートを取得
    const ss = SpreadsheetApp.openById(SPREADSHEET_ID);
    const sheet = ss.getSheetByName(SHEET_NAME);
    
    if (!sheet) {
      console.error("エラー: 指定されたシート名[${SHEET_NAME}]が見つかりません。スプレッドシート内のシート名を確認してください。");
      return;
    }
    
    // 最新の回答を取得
    const lastRow = sheet.getLastRow();
    if (lastRow <= 1) {
      console.log("スプレッドシートに回答データがありません。");
      return;
    }
    
    // 最新行のデータを取得
    // lastRowが最も新しく登録された、sheet.getLastColumn()は最大列
    const range = sheet.getRange(lastRow, 1, 1, sheet.getLastColumn());
    const rowData = range.getValues()[0]; // 1行のデータ取得
    
    console.log("--- スプレッドシートから取得した最新データ ---");
    console.log("取得したデータ: ${rowData}");
    console.log("行データ件数: ${rowData.length}");
    console.log("----------------------------------------------");
    
    // メールアドレスとスコアを取得
    const respondentEmail = rowData[EMAIL_COLUMN_INDEX];
    const totalScore = rowData[SCORE_COLUMN_INDEX];
    
    // 未定義（スプレッドシートから回答したスコアの回答なかった）
    const parsedScore = parseFloat(totalScore);
    
    // メールアドレスが取得できない場合は処理中断する
    if (!respondentEmail) {
      console.error("エラー: 指定された列[${EMAIL_COLUMN_INDEX}]からメールアドレスが取得できませんでした。列インデックスを確認してください。");
      return;
    }
    
    console.log("回答者のメールアドレス: ${respondentEmail}");
    
    // スコアが無効な値であるかチェック
    if (isNaN(parsedScore)) {
      console.error("エラー: スコアが正しく取得できませんでした[${totalScore}]が有効な数値ではありません。SCORE_COLUMN_INDEXの設定を確認してください。");
      return;
    }
    
    console.log("取得したスコア: ${parsedScore}");
    
    // フォームタイトルを設定（カスタマイズ可能）
    const formTitle = "営業基礎研修テスト"; // 必要に応じて変更
    
    let mailBody; // メールの本文
    
    // 合格判定してメールの本文を作成
    if (parsedScore < PASSING_SCORE) {
      // 不合格の場合
      mailBody = 
        `${respondentEmail} 様\n\n` +
        "お疲れ様でした！\n\n" +
        `【今回のスコア: ${parsedScore} 点 / 合格点: ${PASSING_SCORE} 点】\n` +
        "-----------------------------------------\n\n" +
        "残念ながら、今回のテストは合格基準に達しませんでした。\n" +
        "研修資料をもう一度確認の上、以下のリンクから再受験をお願いいたします。\n\n" +
        `📚 研修動画: ${TRAINING_VIDEO_URL}\n` +
        `📝 再受験フォーム: ${FORM_URL}\n\n` +
        "ご不明な点がございましたら、お気軽にお問い合わせください。\n" +
        "よろしくお願いいたします。\n";
    } else {
      // 合格の場合
      console.log("スコアが合格点以上のため、合格メールを送信します。");
      mailBody = 
        `${respondentEmail} 様\n\n` +
        "お疲れ様でした！\n" +
        `【${formTitle}】を受講いただき、ありがとうございます。\n\n` +
        "🎉 合格おめでとうございます！ 🎉\n\n" +
        `あなたのスコア: ${parsedScore} 点\n` +
        `合格点: ${PASSING_SCORE} 点\n` +
        "-------------------------------------\n\n" +
        "研修を修了されました。今後ともよろしくお願いいたします。\n\n" +
        "修了証書は別途お送りいたします。\n";
    }
    
    console.log("合否判定: ${parsedScore >= PASSING_SCORE ? '合格' : '不合格'}");
    
    // メール送信
    console.log("メール送信 ${respondentEmail} に送信しようとしています...");
    MailApp.sendEmail(respondentEmail, EMAIL_SUBJECT, mailBody);
    console.log("メールが送信されました。 ');
    
  } catch (error) {
    // エラー発生時に場合の詳細記録
    console.error("エラー発生しました: ${error.toString()}");
    console.error("スタックトレース: ${error.stack}");
    // 管理者への通知用（メール上で内容についてもエラー原因を確認できます。
    // 例: MailApp.sendEmail("admin@example.com", "GASエラー通知", error.toString());
  }
}
```

## Googleフォームの作成手順

### 1. 基本的なフォーム作成

1. **Googleフォーム**（https://forms.google.com/）にアクセス
2. 「新しいフォームを作成」をクリック
3. タイトルを設定（例：「営業基礎研修テスト」）

### 2. 質問の設定例

```
質問1: メールアドレス
- 種類: 記述式
- 必須回答: ON
- 回答の検証: メールアドレス

質問2: お名前
- 種類: 記述式
- 必須回答: ON

質問3: 営業の第一歩は何ですか？
- 種類: ラジオボタン
- 選択肢: 
  ○ お客様のニーズを聞く（正解）
  ○ 商品の説明をする
  ○ 価格を提示する
  ○ 契約書を準備する

質問4: 効果的なプレゼンテーションのポイントは？
- 種類: チェックボックス
- 選択肢:
  ☑ 相手の立場に立つ（正解）
  ☑ 具体例を示す（正解）
  ☑ 簡潔に伝える（正解）
  ☐ 専門用語を多用する
```

### 3. スプレッドシートとの連携

1. フォーム編集画面で「回答」タブをクリック
2. 「スプレッドシートを作成」ボタンをクリック
3. 新しいスプレッドシートが自動作成されます

## 設定手順

### 1. スプレッドシートIDの設定

```javascript
const SPREADSHEET_ID = "1ABC123xyz789DEF456ghi012JKL345mno678PQR"; // サンプルID
```

**取得方法：**
```
URL例: https://docs.google.com/spreadsheets/d/【ここをコピー】/edit#gid=0
        ↓
ID: 【ここをコピー】 （/d/ と /edit の間の部分）
```

### 2. シート名の確認

```javascript
const SHEET_NAME = "フォームの回答 1"; // 自動生成される標準名
```

### 3. 列インデックスの設定

**スプレッドシートの構造例：**
| A列 | B列 | C列 | D列 |
|-----|-----|-----|-----|
| タイムスタンプ | メールアドレス | お名前 | スコア |
| 2025/1/15 10:30 | test@example.com | 田中太郎 | 85 |

```javascript
const SCORE_COLUMN_INDEX = 3;      // D列（スコア）- 0から始まるので3
const EMAIL_COLUMN_INDEX = 1;      // B列（メールアドレス）- 0から始まるので1
```

### 4. フォームURLと動画URLの設定（推奨）

```javascript
// より確実な方法：直接URLを指定
const FORM_URL = "https://forms.gle/あなたのフォームURL"; // あなたのフォーム短縮URL
const TRAINING_VIDEO_URL = "https://youtu.be/あなたの動画URL"; // 研修動画のURL
```

**フォームURLの取得方法：**
1. Googleフォームで「送信」ボタンをクリック
2. 「リンク」タブを選択
3. 「短縮URL」をコピー

**動画URLの設定：**
- YouTube動画の場合：`https://youtu.be/動画ID`
- 社内動画の場合：社内の動画URL
- 複数の動画がある場合：改行で複数リンクを記載可能

### 5. 合格点とメール内容の設定

```javascript
const PASSING_SCORE = 90;
const EMAIL_SUBJECT = "【受講済】研修営業基礎テストの結果";
```

## 動作フロー

1. **フォーム送信時**: `onFormSubmit()` トリガーが実行
2. **データ取得**: スプレッドシートから最新行のデータを取得
3. **スコア判定**: 合格点と比較して合否を判定
4. **メール送信**: 結果に応じた内容のメールを自動送信

## トリガーの設定

1. Google Apps Script エディタで「トリガー」を選択
2. 「トリガーを追加」をクリック
3. 以下の設定を行う：
   - 実行する関数: `onFormSubmit`
   - イベントのソース: `フォームから`
   - イベントの種類: `フォーム送信時`

### 4. フォームでのスコア設定

**テスト形式にする場合：**
1. フォーム編集画面で「設定」⚙️ をクリック
2. 「テストにする」をONにする
3. 各質問で「回答」→「正解を選択」で正解を設定
4. 各問題の点数を設定

**手動でスコア列を追加する場合：**
1. スプレッドシートに「スコア」列を追加
2. 手動で点数を入力、または数式で計算

**数式でスコア計算する例：**
```
=IF(C2="正解の選択肢", 10, 0) + IF(D2="正解の選択肢", 15, 0) + IF(E2="正解の選択肢", 20, 0)
```

## トラブルシューティング

### よくある問題と解決方法

**1. メールが送信されない**
```javascript
// 権限エラーの場合は、以下でテスト実行
function testEmailSend() {
  MailApp.sendEmail("test@example.com", "テスト", "テストメールです");
}
```

**2. スプレッドシートが見つからない**
```javascript
// シート名を確認
function checkSheetNames() {
  const ss = SpreadsheetApp.openById(SPREADSHEET_ID);
  const sheets = ss.getSheets();
  sheets.forEach(sheet => console.log(sheet.getName()));
}
```

**3. スコアが正しく取得できない**
```javascript
// データ構造を確認
function checkData() {
  const ss = SpreadsheetApp.openById(SPREADSHEET_ID);
  const sheet = ss.getSheetByName(SHEET_NAME);
  const data = sheet.getRange(1, 1, 3, sheet.getLastColumn()).getValues();
  console.log(data);
}
```

**4. フォームURLが正しく取得できない**
- 直接URLを指定する方法（推奨）：
```javascript
const FORM_URL = "https://forms.gle/あなたの短縮URL";
```

## 注意点

- **権限の設定**: 初回実行時にメール送信とスプレッドシート読み取りの権限許可が必要
- **テスト実行**: 本番使用前に必ずテストを行ってください
- **エラーハンドリング**: エラー発生時はログを確認し、設定値を見直してください
- **重複送信**: 同じ回答に対して複数回メールが送信される可能性があるため、必要に応じて重複チェック機能を追加してください
