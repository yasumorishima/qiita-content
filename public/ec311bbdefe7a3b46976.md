# 【Selenium】画像無効化設定で画像が表示されなくなった時の完全回復方法

## 🚨 問題の発生

Seleniumで**高速化のために画像を無効化**したところ、設定を元に戻しても画像が表示されなくなってしまいました。

### 問題の原因となったコード

```python
# 高速化のために画像を無効化（これが問題の原因）
prefs = {
    "profile.managed_default_content_settings.images": 2,  # ← この設定が悪さ
    "profile.default_content_settings.popups": 0,
    "profile.default_content_setting_values.notifications": 2
}
options.add_experimental_option("prefs", prefs)
```

## 😰 設定を元に戻しても画像が復活しない

コードから画像無効化の設定を削除しても、画像は表示されませんでした。

```python
# コードを修正（でも効果なし）
prefs = {
    # "profile.managed_default_content_settings.images": 2,  # ← コメントアウト
    "profile.default_content_settings.popups": 0,
    "profile.default_content_setting_values.notifications": 2
}
```

### さらに画像有効化を明示的に設定しても効果なし

```python
# 明示的に有効化を試行（でも効果なし）
prefs = {
    "profile.managed_default_content_settings.images": 1,  # ← 有効化に変更
    "profile.default_content_settings.popups": 0,
    "profile.default_content_setting_values.notifications": 2
}
```

## 🔍 原因の調査

### なぜ画像が復活しないのか？

Seleniumで一度画像無効化設定を行うと、**ブラウザのプロファイル設定ファイルに保存**されてしまいます。その後、コードで設定を変更しても、**保存された設定が優先**されるため画像が表示されません。

### 設定が保存される場所

- **Chrome**: `C:\Users\{ユーザー名}\AppData\Local\Google\Chrome\{プロファイル名}\Default\Preferences`
- **Brave**: `C:\Users\{ユーザー名}\AppData\Local\BraveSoftware\Brave-Browser\{プロファイル名}\Default\Preferences`

## 💡 試した解決方法（効果なし）

### 1. ブラウザのキャッシュクリア
- 設定 → プライバシーとセキュリティ → 閲覧履歴データを削除
- **結果**: 効果なし

### 2. より強力なprefs設定
```python
# 複数箇所で画像設定を試行（効果なし）
prefs = {
    "profile.managed_default_content_settings.images": 1,
    "profile.default_content_settings.images": 1,
    "profile.content_settings.exceptions.images": {"*": {"setting": 1}},
    "profile.default_content_setting_values.images": 1
}
```
- **結果**: 効果なし

### 3. ブラウザ設定ページでの手動変更
- `chrome://settings/content/images` で手動設定
- **結果**: 一時的に有効化されるが、Selenium実行時に再び無効化

## ✅ 最終解決方法：プロファイル設定ファイルの直接編集

**プロファイル設定ファイル（Preferences）を直接編集**することで、確実に画像表示を復活させることができました。

### Chrome用 回復スクリプト

```python
import os
import json
import shutil
from datetime import datetime

def force_enable_images_chrome():
    """Chrome Selenium専用プロファイルの画像設定を強制有効化"""
    try:
        # プロファイルパス（適宜変更してください）
        profile_path = r'C:\Users\{ユーザー名}\AppData\Local\Google\Chrome\SeleniumProfile\Default'
        preferences_file = os.path.join(profile_path, 'Preferences')
        
        print("🔧 Chrome プロファイル設定ファイルを編集中...")
        
        # バックアップ作成
        if os.path.exists(preferences_file):
            timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
            backup_file = f"{preferences_file}.backup_{timestamp}"
            shutil.copy2(preferences_file, backup_file)
            print(f"💾 バックアップ作成: {backup_file}")
            
            # 既存設定を読み込み
            with open(preferences_file, 'r', encoding='utf-8') as f:
                prefs_data = json.load(f)
        else:
            prefs_data = {}
        
        # profile セクションの確保
        if 'profile' not in prefs_data:
            prefs_data['profile'] = {}
        
        profile = prefs_data['profile']
        
        # 画像設定を強制的に有効化
        if 'managed_default_content_settings' not in profile:
            profile['managed_default_content_settings'] = {}
        profile['managed_default_content_settings']['images'] = 1
        
        if 'default_content_settings' not in profile:
            profile['default_content_settings'] = {}
        profile['default_content_settings']['images'] = 1
        
        if 'default_content_setting_values' not in profile:
            profile['default_content_setting_values'] = {}
        profile['default_content_setting_values']['images'] = 1
        
        # ディレクトリ作成
        os.makedirs(profile_path, exist_ok=True)
        
        # 安全な書き込み
        temp_file = f"{preferences_file}.tmp"
        with open(temp_file, 'w', encoding='utf-8') as f:
            json.dump(prefs_data, f, indent=2, ensure_ascii=False)
        shutil.move(temp_file, preferences_file)
        
        print("✅ 画像設定の有効化完了")
        return True
        
    except Exception as e:
        print(f"❌ エラー: {e}")
        return False

if __name__ == "__main__":
    force_enable_images_chrome()
```

## 🛡️ 予防策

### 今後画像無効化を避ける方法

```python
# 画像無効化の代わりに、他の高速化方法を使用
options = webdriver.ChromeOptions()

# 軽量化オプション（画像は有効のまま）
options.add_argument('--disable-logging')
options.add_argument('--disable-extensions')
options.add_argument('--disable-gpu')
options.add_argument('--no-sandbox')

# 通知やポップアップのみ無効化
prefs = {
    "profile.default_content_settings.popups": 0,
    "profile.default_content_setting_values.notifications": 2
    # 画像設定は含めない
}
options.add_experimental_option("prefs", prefs)
```

### 画像無効化が必要な場合の対策

```python
# 専用プロファイルを使用して影響を限定
options.add_argument('--user-data-dir=C:\\temp\\selenium_no_images')  # 使い捨てプロファイル
```

## 📝 学んだ教訓

1. **Seleniumの設定は永続化される**: 一度設定すると、プロファイルに保存される
2. **コード修正だけでは不十分**: 保存された設定が優先される
3. **プロファイル設定ファイルの直接編集が最も確実**: 最終手段として有効
4. **バックアップの重要性**: 設定変更前は必ずバックアップを取る

## 🔗 参考情報

### プロファイル設定ファイルの場所

| ブラウザ | パス |
|---------|------|
| Chrome | `%LOCALAPPDATA%\Google\Chrome\{プロファイル名}\Default\Preferences` |
| Brave | `%LOCALAPPDATA%\BraveSoftware\Brave-Browser\{プロファイル名}\Default\Preferences` |

### 設定ファイルの構造

```json
{
  "profile": {
    "managed_default_content_settings": {
      "images": 1  // 1=有効, 2=無効
    },
    "default_content_settings": {
      "images": 1
    },
    "default_content_setting_values": {
      "images": 1
    }
  }
}
```

予防策として：
