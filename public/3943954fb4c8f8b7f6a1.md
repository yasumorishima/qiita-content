## 🔥 MySalesData：完全無料の挑戦者

### 無料で全機能？本当に大丈夫？

**開発元：** MSD Development Team 
**価格：** 完全無料（有料プランなし）  
**評価：** 4.0/5（Google Workspace Marketplace）

「タダより高いものはない」という言葉がありますが、MySalesDataは本当に無料で以下の機能を提供しています：

✅ **2000行制限なしでデータ取得**  
✅ **自動スケジューラー機能**  
✅ **REST API & SOAP API両対応**  
✅ **データ変更時のメール通知**  
✅ **双方向データ同期**

「なんで無料でこんなにできるの？」と疑問に思いますよね。

### ただし、重要な「But」があります

**SOQLスキルが必須：**
MySalesDataは、SOQL（Salesforce Object Query Language）の知識が前提になっています。

例えば、「過去30日の商談データを取得したい」場合：
```sql
SELECT Id, Name, Amount, CloseDate, StageName 
FROM Opportunity 
WHERE CloseDate >= LAST_N_DAYS:30
```

こういったクエリを書けないと、MySalesDataの真価を発揮できません。

**サポートが限定的：**
- ドキュメントは基本的なもののみ
- コミュニティサポートが中心
- 最新アップデートは2019年（5年以上前）

## ❓ よくある質問

**Q: 本当に2000行制限を回避できるの？**
A: はい、両ツールともSOQLを使用するため、Salesforce Report APIの2000行制限を受けません。実際に10万行以上のデータ取得に成功しています。

**Q: セキュリティは大丈夫？**
A: G-ConnectorはGDPR準拠、第三者監査済み。MySalesDataも基本的なOAuth認証を使用していますが、詳細なセキュリティ情報は限定的です。

**Q: 初心者でも使える？**
A: G-Connectorは初心者フレンドリー。MySalesDataはSQLの知識が必要なため、学習コストがあります。

**Q: 移行は簡単？**
A: G-Connectorは14日間の無料トライアル、MySalesDataは永続無料なので、リスクなく試せます。
