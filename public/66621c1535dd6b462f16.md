# ICPの現在価格(USD建て、JPY建て)を取得し、Lineに送る

必要なAPI

*   coinmarketcap API<br><https://coinmarketcap.com/api/>

*   Line notify<br><https://notify-bot.line.me/ja/>

***

# GASで実施

*   GAS<br><https://script.google.com/home>

GAS は Google Apps Script の略です。Google Apps Script は、Google のサービス (Google Spreadsheet, Google Form, Gmail, Google Drive, Google Calendar, etc.) で簡単なスクリプトを作成、実行、共有することができるプラットフォームです。JavaScript 言語を使用して、Web アプリケーションを開発することもできます。Google Apps Script により、Google のサービスを統合することができ、効率的なワークフローを作成することができます。　※ChatGPTより

## 価格を取得

このスクリプトは、API キーを使用して CoinMarketCap の API から特定の暗号資産 (ICP) の現在の価格を取得し、その価格を米ドルと日本円で表示するもの

<pre><code>var API_KEY = "ここはCoinMarketCapで取得したAPIを書く";

function getCryptoPrice() {
  var responseJPY = UrlFetchApp.fetch("https://pro-api.coinmarketcap.com/v1/cryptocurrency/quotes/latest?symbol=ICP&convert=JPY&CMC_PRO_API_KEY=" + API_KEY);
  var jsonJPY = JSON.parse(responseJPY.getContentText());
  var price_jpy = jsonJPY.data.ICP.quote.JPY.price;

  var responseUSD = UrlFetchApp.fetch("https://pro-api.coinmarketcap.com/v1/cryptocurrency/quotes/latest?symbol=ICP&convert=USD&CMC_PRO_API_KEY=" + API_KEY);
  var jsonUSD = JSON.parse(responseUSD.getContentText());
  var price_usd = jsonUSD.data.ICP.quote.USD.price;

  Logger.log("Price in USD: " + price_usd);
  Logger.log("Price in JPY: " + price_jpy);
}
</code></pre>

## Lineに送る

このスクリプトは、CryptoCurrency APIから指定の暗号資産（この場合、ICP）の価格を取得し、その価格をLine Notifyに通知するものです。APIキーとLine Notifyのアクセストークンが必要なことに注意

<pre><code>var API_KEY = "ここはCoinMarketCapで取得したAPIを書く";
var LINE_NOTIFY_ACCESS_TOKEN = "your_line_notify_access_token";

function getCryptoPrice() {
  var responseJPY = UrlFetchApp.fetch("https://pro-api.coinmarketcap.com/v1/cryptocurrency/quotes/latest?symbol=ICP&convert=JPY&CMC_PRO_API_KEY=" + API_KEY);
  var jsonJPY = JSON.parse(responseJPY.getContentText());
  var price_jpy = jsonJPY.data.ICP.quote.JPY.price;

  var responseUSD = UrlFetchApp.fetch("https://pro-api.coinmarketcap.com/v1/cryptocurrency/quotes/latest?symbol=ICP&convert=USD&CMC_PRO_API_KEY=" + API_KEY);
  var jsonUSD = JSON.parse(responseUSD.getContentText());
  var price_usd = jsonUSD.data.ICP.quote.USD.price;

  Logger.log("Price in USD: " + price_usd.toFixed(2));
  Logger.log("Price in JPY: " + price_jpy.toFixed(2));

  var message = "ICP Price in USD: " + price_usd.toFixed(2) + "\nPrice in JPY: " + price_jpy.toFixed(2);
  
  var options = {
    method: "post",
    headers: {
      "Authorization": "Bearer " + LINE_NOTIFY_ACCESS_TOKEN
    },
    payload: {
      "message": message
    }
  };
  
  var response = UrlFetchApp.fetch("https://notify-api.line.me/api/notify", options);
  Logger.log("Response: " + response.getContentText());
}</code></pre>
