## 改修版での変更点

### 主な変更

1. **faster-whisperへの変更** - 元のwhisper-openaiからfaster-whisperに変更。処理速度が向上
2. **GPU対応の強化** - CUDAの自動検出とメモリ管理を追加
3. **モデルサイズの自動切り替え** - large-v3でメモリ不足が起きた時、自動的にmediumに切り替え
4. **VADフィルター追加** - 無音区間を検知して除外し、処理速度を向上
5. **タイムスタンプ付きプレビュー** - 処理中に結果をリアルタイムで確認可能

### 必要なモジュールのインポート

```python
import os
import time
import torch
import gc
from google.colab import files
from pydub import AudioSegment
from faster_whisper import WhisperModel
```

### メイン処理

```python
# デバイスの確認
device = "cuda" if torch.cuda.is_available() else "cpu"
if device == "cpu":
    print("[警告] GPUが認識されていません。")
    print("メニューの [ランタイム] → [ランタイムのタイプを変更] から [T4 GPU] を選択してください。")
else:
    print(f"使用中のデバイス: {device}")

# メモリのクリーンアップ
gc.collect()
if torch.cuda.is_available():
    torch.cuda.empty_cache()

# モデルの読み込み
model_size = "large-v3"
print(f"Whisperモデル ({model_size}) を読み込んでいます...")

try:
    model = WhisperModel(model_size, device=device, compute_type="float16" if device == "cuda" else "int8")
    print(f">> {model_size} モデルの読み込みに成功しました。")
except Exception as e:
    print(f"Largeモデルの読み込みに失敗しました。Mediumに切り替えます。エラー: {e}")
    model = WhisperModel("medium", device=device, compute_type="float16" if device == "cuda" else "int8")
    print(">> Mediumモデルの読み込みに成功しました。")

# ファイルのアップロード
print("\n文字起こししたい音声ファイルをアップロードしてください。")
uploaded = files.upload()

if not uploaded:
    print("ファイルがアップロードされませんでした。終了します。")
else:
    for filename in uploaded.keys():
        print(f"\n{'='*50}")
        print(f"処理開始: {filename}")
        print(f"{'='*50}")
        
        # 音声の事前処理
        print(">> 音声を最適化中 (16kHz / Mono / Normalize)...")
        try:
            audio = AudioSegment.from_file(filename)
            audio = audio.set_frame_rate(16000).set_channels(1).normalize()
            
            temp_wav = "temp_processing.wav"
            audio.export(temp_wav, format="wav")
            
            del audio
            gc.collect()
        except Exception as e:
            print(f"音声ファイルの処理中にエラーが発生しました: {e}")
            continue
        
        # 文字起こし実行
        start_time = time.time()
        print(">> 文字起こし中 (日本語固定モード) ...")
        
        segments, info = model.transcribe(
            temp_wav,
            language="ja",
            initial_prompt="これは日本語の会議の議事録です。えー、あの、などのフィラーは適宜省略してください。",
            beam_size=5,
            vad_filter=True,
            vad_parameters=dict(min_silence_duration_ms=500)
        )
        
        # 結果の処理
        results = []
        print("\n--- 文字起こし内容（プレビュー） ---")
        for segment in segments:
            timestamp = f"[{segment.start:05.1f}s -> {segment.end:05.1f}s]"
            print(f"{timestamp} {segment.text}")
            results.append(segment.text)
        
        full_text = "\n".join(results)
        
        end_time = time.time()
        print(f"\n>> 完了！ 処理時間: {(end_time - start_time):.2f}秒")
        
        # 結果の保存
        base_name = os.path.splitext(filename)[0]
        output_filename = f"transcription_{base_name}.txt"
        
        with open(output_filename, "w", encoding="utf-8") as f:
            f.write(full_text)
        
        print(f">> 保存完了: {output_filename}")
        files.download(output_filename)
        
        # 一時ファイルの削除
        if os.path.exists(temp_wav):
            os.remove(temp_wav)
    
    print("\nすべてのファイルの処理が終了しました。")
```

## 改修のポイント

### 1. faster-whisperの採用

元のコードではwhisper-openaiを使用していたが、faster-whisperに変更した。これにより処理速度が大幅に向上した。

**変更前:**
```python
import whisper
model = whisper.load_model("medium")
result = model.transcribe(wav_filename, language="ja")
```

**変更後:**
```python
from faster_whisper import WhisperModel
model = WhisperModel(model_size, device=device, compute_type="float16")
segments, info = model.transcribe(temp_wav, language="ja", vad_filter=True)
```

### 2. GPU対応の強化

CUDAが使用可能かどうかを自動検出し、GPUが使えない場合は警告を表示する。また、メモリのクリーンアップ処理を追加した。

```python
device = "cuda" if torch.cuda.is_available() else "cpu"
gc.collect()
if torch.cuda.is_available():
    torch.cuda.empty_cache()
```

### 3. モデルサイズの自動切り替え

large-v3モデルでメモリ不足が発生した場合、自動的にmediumモデルに切り替える。これにより、エラーで処理が止まることを防ぐ。

```python
try:
    model = WhisperModel("large-v3", device=device, compute_type="float16")
except Exception as e:
    print(f"Largeモデルの読み込みに失敗しました。Mediumに切り替えます。")
    model = WhisperModel("medium", device=device, compute_type="float16")
```

### 4. VADフィルターの追加

VAD（Voice Activity Detection）フィルターを使用することで、無音区間を検知して除外する。これにより処理速度が向上し、不要な「無音」の認識を防ぐ。

```python
segments, info = model.transcribe(
    temp_wav,
    language="ja",
    vad_filter=True,
    vad_parameters=dict(min_silence_duration_ms=500)
)
```

### 5. タイムスタンプ付きプレビュー

処理中に各セグメントのタイムスタンプと内容を表示することで、リアルタイムで結果を確認できる。

```python
for segment in segments:
    timestamp = f"[{segment.start:05.1f}s -> {segment.end:05.1f}s]"
    print(f"{timestamp} {segment.text}")
```

## まとめ

faster-whisperへの変更により、処理速度と安定性が向上した。GPU対応の強化とメモリ管理の改善により、より大きなファイルでも安定して処理できる。VADフィルターとタイムスタンプ付きプレビューにより、使い勝手も向上した。

音量が小さい音声でも、normalize()による正規化は引き続き有効。NotebookLMで文字起こしできない場合の代替手段として活用できる。

## 注意事項

- GPU設定を有効にすること（処理速度が大幅に向上する）
- large-v3モデルは高精度だが、メモリを多く使用する
- 長時間の音声ファイルはセッションタイムアウトに注意
