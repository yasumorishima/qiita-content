'2023-04-21', '2023-04-27', '2023-05-03', '2023-05-09',

'2023-08-03', '2023-08-09', '2023-08-23'
]


# Create an empty DataFrame to store the data
df_660271_all_dates = pd.DataFrame()

# Fetch data for each date and concatenate
for date in dates:
 df_single_day = statcast(start_dt=date, end_dt=date)

 df_660271_all_dates = pd.concat([df_660271_all_dates, df_660271_single_day])

# Reset the index of the final DataFrame
df_660271_all_dates.reset_index(drop=True, inplace=True)

# 各グラフ

## スピンレート / 各球種

`pybaseball`における「スピンレート（spin_rate）」は、ピッチが放たれた際にボールが回転する速度を示します。この数値は、分毎の回転数（rpm、revolutions per minute）で表されます。スピンレートが高いほど、ボールはより多くの回転をし、これがピッチの動きに影響を与えます。例えば、スピンレートが高いファストボールは「上昇」するように見えることがあり、カーブボールやスライダーのような変化球はより鋭い動きをすることがあります。スピンレートは、ピッチの効果を分析し、ピッチャーのスキルを評価するのに役立つ重要な指標です。

## リリースエクステンション / 各球種

`pybaseball`での「リリースエクステンション（release_extension）」は、ピッチャーの手からボールが離れるポイントからホームプレートまでの距離を表します。これは、ボールがピッチャーの手を離れてからホームプレートに到達するまでの物理的な距離をメートルまたはフィートで示す数値です。リリースエクステンションが長いほど、ピッチャーはホームプレートに近い位置でボールをリリースしていることを意味し、これはしばしばバッターにとって反応時間が短くなることを意味します。

## release_pos_x / 各球種

`pybaseball`での「release_pos_x」は、ピッチャーがボールをリリースする位置の水平面上の座標を示します。この値は、ピッチャーマウンドの中央から左右にどれだけ離れてボールがリリースされたかを示し、フィートまたはメートルで測定されます。

リリースポイントはピッチャーごとに異なり、そのポイントはピッチャーの投球スタイルや身体の動きに影響を受けます。例えば、右投げのピッチャーであれば、`release_pos_x`の値が正の場合、彼はマウンドの右側（ファーストベース側）でボールをリリースしていることを意味します。逆に、左投げのピッチャーであれば、この値が負の場合、彼はマウンドの左側（サードベース側）でボールをリリースしていることを意味します。

## effective_speed / 各球種

`pybaseball`における「effective_speed」は、ピッチの実効速度を表します。これは、ピッチャーのリリースポイントの距離とリリーススピードを考慮に入れた、バッターが感じるピッチの速度です。

ピッチの実効速度は、単にピッチャーの手から離れた時の速度（リリーススピード）よりも、バッターの視点から見たピッチの速度をより正確に反映します。ピッチャーがホームプレートに近い位置でボールをリリースする場合、実効速度はリリーススピードよりも速く感じられることがあります。逆に、ピッチャーが比較的遠い位置からボールをリリースする場合、実効速度はリリーススピードよりも遅く感じられることがあります。

## vx0 / 各球種

`pybaseball`における「vx0」は、ピッチがリリースされた瞬間のボールの水平方向の速度成分を表します。この値は、ボールがピッチャーの手から離れた後、ホームプレートに向かって左右どちらの方向に移動しているかを示し、フィート毎秒またはメートル毎秒で測定されます。

## vx0 / 各球種

# √(release_pos_x^2 + release_pos_z^2)"とrelease_spin_rate　の積をplot

故障予知の観点で確認

「√(release_pos_x^2 + release_pos_z^2)」と「release_spin_rate」の積は、ピッチャーのリリースポイントの位置とボールのスピンレートを組み合わせた指標を表します。具体的には、次の二つの要素を組み合わせたものです：

1.  **リリースポイントの距離**: 「√(release_pos_x^2 + release_pos_z^2)」は、ピッチャーのリリースポイントの地面からの距離を表します。ここで、`release_pos_x`はリリースポイントの水平方向の位置（左右）、`release_pos_z`は垂直方向の位置（上下）です。この距離は、ピッチャーがどれだけ高い位置から、またはどれだけ横に広がった位置からボールをリリースしているかを示します。

2.  **スピンレート**: 「release_spin_rate」は、ボールがピッチャーの手から離れる瞬間の回転速度（rpm）を示します。スピンレートは、ボールの軌道や動きに大きく影響します。

これら二つの要素を掛け合わせることで、ピッチャーのリリースポイントの物理的な位置とボールの回転特性の組み合わせが得られ、これがピッチの全体的な軌道や効果にどのように影響するかを理解するための一つの方法となります。
