## 発見したこと（結論から）

**腕の速さがほぼ同じ（24〜26 m/s）投手の間で、球速が最大13mph（約20km/h）違う。**

この差は「腕の力」では説明できない。では何が違うのか？

61投手分のプロレベルモーションキャプチャデータを分析したところ、**「体をどう使うか」を表す5つの要因**が球速の差を説明することが分かりました。

→ **GitHub**: https://github.com/yasumorishima/baseball-cv

---

## 使ったデータ

**Driveline OpenBiomechanics Project（OBP）**

- 対象: 61投手分のC3Dモーションキャプチャデータ
- マーカー数: 体全体45箇所（肩・肘・手首・腰・膝・かかとなど）
- サンプリングレート: 360Hz（1秒間に360回計測）
- 球速範囲: 71.3〜93.1 mph（約115〜150 km/h）

C3Dファイルの読み込みには [ezc3d](https://github.com/pyomeca/ezc3d) を使用しています。

---

## 「体効率」という指標を作る

まず前提として、**球速と腕速度には強い相関**（r=0.67）があります。速い腕を持つ投手は速い球を投げる、という当たり前の話です。

ここで「腕速度から期待される球速との差」を計算します。

```python
# 腕速度(peak_wrist_linear_speed)から球速を線形予測
lm = LinearRegression().fit(df[['peak_wrist_linear_speed']], df['pitch_speed_mph'])

# 実際の球速 − 予測球速 = "体効率"
df['body_efficiency'] = df['pitch_speed_mph'] - lm.predict(df[['peak_wrist_linear_speed']])
```

`body_efficiency` がプラスなら「腕速度のわりに球が速い＝体をうまく使っている」、マイナスなら「腕は速いのに球が出ない＝腕に頼っている」投手です。

この指標で投手を5グループ（Q1〜Q5）に分けました。Q1が「腕の割に球速が遅い（非効率）」、Q5が「腕の割に球速が速い（効率的）」グループです。

## グラフ1: 全体像

![体効率の全体像](https://raw.githubusercontent.com/yasumorishima/baseball-cv/master/data/output/efficient_throwing_story.png)

このグラフは3つのパネルで構成されています。

**左パネル（散布図）**: 横軸が腕速度（m/s）、縦軸が球速（mph）。同じ腕速度（縦の並び）でも球速がばらついているのが見えます。これが今回のテーマです。

**中央パネル（R²の積み上げ）**: 各要因を1つずつ追加したときに、モデルがどれだけ球速を説明できるようになるかを示しています。棒グラフが高くなるほど「説明力が上がった」ということです。

**右パネル（Q1 vs Q5 比較）**: 体効率の低いグループ（Q1）と高いグループ（Q5）で、各指標の「平均からのずれ」を比較しています（0=平均と同じ、プラス=球速向上に有利、マイナス=不利）。棒が長いほど2グループの差が大きいことを意味します。

---

## 5つの要因と、それぞれの意味

| 要因 | R² | 物理的な意味 |
|---|---|---|
| 腕速度 + 身長 | 0.473 | ベースライン |
| + ストライド（並進） | 0.477 | 踏み出し幅：体の前進量 |
| + 脚引き上げ（弾性） | 0.522 | 踏み出し前の膝の高さ |
| + 腕連鎖（whip） | 0.562 | 体が肘を駆動しているか |
| + **膝スムーズさ** | **0.648** | 着地脚の動きの滑らかさ（最大寄与） |

**R²=0.648** は、これら5要因で球速のばらつきの約65%を説明できるということです。腕速度だけでは約47%だったのが18ポイント改善しました。

### 各要因の解説

**ストライド**: 踏み出し足が着地するまでにどれだけ前に進むか。長いほど体重移動が大きく、並進エネルギーが腕に伝わりやすい。

**脚引き上げ**: 投球前の膝の高さ。高く引き上げるほど股関節の弾性エネルギーが蓄積され、踏み出し時のパワーが増す。

**腕連鎖（whip）**: 「肘のスピード ÷ 手首のスピード」の比率。低いほど体が肘を引っ張って加速させている状態（体主導）。

**膝スムーズさ**: 着地脚（リード脚）の膝の動きの「なめらかさ」を数値化したものです。膝の3D軌跡のぶれ（動きの不規則さ）を計算し、値が低いほど膝が滑らかに動いていることを示します。

---

## 「膝スムーズさ」はなぜ効くのか

これには面白い統計的な現象があります。

- 全体で見ると: r=+0.12（腕が速い投手は全身の動きが激しく、膝のぶれも大きくなる傾向）
- 腕速度が同じ条件に絞ると: r=−0.45***（腕速度の差を除外すると、膝がなめらかな投手ほど球速が速い）

つまり「腕の速さの影響を取り除いて初めて、膝のなめらかさの効果が見えてくる」指標です。

メカニズムの仮説：**スムーズな膝 → 骨盤が効率よく回転 → 骨盤/腕速度比が向上（約17%差） → 体が腕を「引っ張る」連鎖が効く**

## 骨格GIF: 同じ腕速度で10mph差

![Q1 vs Q5 骨格比較](https://raw.githubusercontent.com/yasumorishima/baseball-cv/master/data/output/efficient_thrower_comparison.gif)

**左（Q1）**: 腕速度26.56 m/s → **80.8 mph**（ストライド0.30m）
**右（Q5）**: 腕速度24.96 m/s → **91.8 mph**（ストライド0.89m）

赤い部分がリード脚（踏み出し足）です。右のQ5投手は明らかに長い踏み出しをしており、体全体で投げていることが分かります。オレンジの星マークは着地点です。

## 根本原因：なぜQ1はストライドが短いのか

着地直前の足首の動き（**ankle braking**）に注目します。

- Q1: 足首ブレーキ ≈ 0.06 m/s²（ほぼブレーキなし）
- Q5: 足首ブレーキ ≈ 3.58 m/s²（強いブレーキ）

ankle_braking → stride の相関: **r=+0.55***

さらに膝の引き上げにも差がありました：
- Q1: 膝の最大屈曲角 85.4°（引き上げが浅い）
- Q5: 膝の最大屈曲角 76.0°（引き上げが深い）

連鎖のまとめ：

```
膝の引き上げ不足 → ストライド短い
足首ブレーキ弱い → ストライド短い
              ↓
         体重移動が小さい → 腕の連鎖が弱い → 球速低下
```

---

## まとめ

- 腕速度が同じ投手の間で球速が最大13 mph異なる
- 「体効率」で分類すると、5つの要因が球速差を説明（R²=0.648）
- 最も説明力が高かったのは「膝スムーズさ」（腕速度の影響を除外して初めて効果が現れる）
- 根本原因は足首ブレーキ・膝引き上げ → ストライドを通じた連鎖的影響
