## はじめに

「バンテリンドームは投手天国」「神宮は打者有利」——野球ファンなら感覚的に知っていること。でも、それを**数値で証明**したことはあるでしょうか。

NPBの12球場はサイズも天然芝の有無も照明も全然違います。ホームランが出やすい球場もあれば、フライが犠牲フライになる球場もある。これを無視して「阪神の投手陣はすごい」と言っても、それは球場の力かもしれない。

今回は2016〜2025年の8,619試合スコアから**パークファクター(PF)**を計算し、12球団の年度推移を可視化しました。さらに、改修工事がPFをどう変えたかも分析します。

→ **GitHub**: https://github.com/yasumorishima/npb-prediction

---

## パークファクターとは

パークファクターは「この球場で試合すると、どれだけ得点が増えるか/減るか」を表す指標です。

### 計算式（Baseball Reference標準）

```
PF = ((ホーム得点 + ホーム失点) / ホーム試合数)
   / ((アウェイ得点 + アウェイ失点) / アウェイ試合数)

PF > 1.00 : 打者有利（点が入りやすい）
PF = 1.00 : 中立
PF < 1.00 : 投手有利（点が入りにくい）
```

### なぜ「ホーム得点だけ」ではダメか

```python
# ❌ チームの打力が混入してしまう
pf_bad = home_score / average_score

# ✅ 同じチームのホームvsアウェイを比較することで球場効果のみを抽出
pf_good = (home_RS + home_RA) / home_G / ((away_RS + away_RA) / away_G)
```

「ホームとアウェイで同じチームがどれだけ違う環境で戦っているか」を見ることで、球場の純粋な影響だけを取り出せます。

### FanGraphs方式（5年平均）の意味

1年だけだと乱数の影響が大きい（特にHRは少ないサンプルで振れやすい）。そこで**3〜5年の合算データ**を使うのが一般的。今回は5年平均（PF_5yr）も計算し、折れ線グラフで表示しています。

## データ取得

npb.jp の試合スコアを `requests` + `BeautifulSoup` でスクレイピング。

```python
import requests
from bs4 import BeautifulSoup

url = f"https://npb.jp/scores/{year}/index.html"
res = requests.get(url, timeout=10)
res.encoding = "utf-8"  # ← charset=UTF-8 でも明示指定が必要
```

:::note warn
**requestsのUTF-8罠**: npb.jpのHTMLはUTF-8ですが、`res.encoding` を指定しないと `apparent_encoding`（Shift-JIS推定）に引っ張られて文字化けします。必ず明示指定してください。
:::

## 5年平均の落とし穴：改修をまたいだデータは意味を失う

**例: バンテリンドーム(中日)**

2025年のPF_5yr=0.844。ところが2026年にHRウイング設置（左右中間▲6m）が予定されています。改修後も5年前のデータを混ぜ続けたら、**2030年頃まで実態より低いPFが表示され続けてしまう**。

これを避けるため「改修ブレイクポイント」を実装。改修年以降のデータのみを5年平均計算に使います。


## 注目球場の考察

### ① PayPayドーム（ソフトバンク）── HRテラスで中立球場へ

2015年のHRテラス設置（左右中間▲6m）直後の2016年はPF=1.171。しかし10年で徐々に収束し、2025年のPF_5yr=1.007とほぼ中立になっています。

### ② ZOZOマリン（ロッテ）── 2019年改修で投手天国→打者有利へ

| 年度 | PF | 備考 |
|---|---|---|
| 2018 | 0.874 | 改修前 |
| 2019 | 0.923 | HRラグーン設置 |
| 2020 | 1.101 | 翌年、急上昇 |
| 2021 | 1.235 | 打者有利化 |
| 2025 | 1.010 | PF_5yr=1.097 |

改修翌年から劇的に打者有利へ転換。フェンス変更が即座に数字に表れた好例です。

### ③ エスコンフィールド（日本ハム）── 新球場は2年かけて本領発揮

| 年度 | PF | PF_5yr |
|---|---|---|
| 2022（札幌ドーム） | 0.949 | 0.967 |
| 2023（1年目） | 0.969 | 0.969 |
| 2024（2年目） | 1.212 | 1.089 |
| 2025（3年目） | 1.271 | 1.147 |

初年度は右中間が広い影響かPF=0.969でしたが、2年目以降は1.2超の打者有利。選手がコースを把握するまで時間がかかった、という見方もできます。

### ④ バンテリンドーム（中日）── 10年間の投手天国、2026年に変わるか

10年間を通じてPF_5yr=0.773〜0.955の投手天国。2025年はやや改善してPF=0.955ですが、それでも平均より1割以上点が入りにくい。

2026年にHRウイング設置（左右中間▲6m）が予定されており、ZOZOマリン2019年と同様の変化が起きる可能性があります。

### ⑤ 楽天モバイルパーク（楽天）── 2回の改修、2026年が本命

2016年の天然芝化（左中間フェンス低下）と2026年のフェンス前方移設が予定。現在PF_5yr=0.908の投手有利から中立〜打者有利への変化に注目です。

## 2026年展望

バンテリン・楽天の2球場で大きな改修が予定されており、特に**中日と楽天の投手陣の数字の見え方**に影響します。防御率が上がっても球場変更が原因かもしれない、という視点が重要です。
